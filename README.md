# poo-animales-python

Ejercicios de POO en Python: clases, herencia y polimorfismo aplicados a animales.

## Conceptos aplicados
- **Clases y atributos** — clase base `Animal` con `nombre` y `especie`
- **Herencia** — cada animal es una subclase de `Animal`
- **Polimorfismo** — cada subclase tiene su propio `hacer_sonido()`
